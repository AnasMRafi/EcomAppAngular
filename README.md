# E-Commerce Angular Application

A full-stack e-commerce application with an Angular frontend and a Spring Boot products microservice.

## Project Structure

```
ecom-angular/
├── frontend/          # Angular 19 frontend application
├── products-service/  # Spring Boot backend microservice
└── screenshots/       # Application screenshots
```

## Frontend

The Angular frontend is located in the `frontend/` directory.

### Running the Frontend

```bash
cd frontend
npm install
ng serve
```

The application will be available at `http://localhost:4200`.

## Backend (Products Service)

The Spring Boot products microservice is located in the `products-service/` directory.

### Running the Backend

```bash
cd products-service
mvn spring-boot:run
```
